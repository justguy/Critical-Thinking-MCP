Beta 2 Review Surface

Interactive review for the release-gate benchmark.

This page turns the Beta 2 release-gate run into a reviewer-facing control surface. It reads the real benchmark JSON, computes the B-arm control-plane summary, and lets you inspect where the orchestrator bounded, salvaged, or escalated each case.

If this page is opened as a local file and the browser blocks JSON fetches, serve the repo with python3 -m http.server 8000 and open the page through http://localhost:8000/html/ct-beta2-benchmark-review.html.
